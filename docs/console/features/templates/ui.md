## 交互状态

| 状态         | 表现                                                         |
| ------------ | ------------------------------------------------------------ |
| **加载中**   | 列表页骨架屏（2 个卡片占位）                                 |
| **空状态**   | `src/presets/` 下无模板时，显示"暂无可用模板"+ 说明文字      |
| **安装中**   | 按钮变为 loading 状态（spinner + "部署中..."），禁止重复点击 |
| **安装成功** | Toast 提示"模板部署成功"，列表页刷新，状态变为「已激活」     |
| **安装失败** | 红色 Alert 显示具体错误（如"SOUL.md 缺失：bingbu"）          |
| **已激活**   | 卡片右上角绿色「已激活」Badge，安装按钮变为「重新部署」      |

## 响应式

| 断点 | 列表页                             | 详情页                      |
| ---- | ---------------------------------- | --------------------------- |
| 手机 | 卡片全宽，按钮堆叠为纵向           | Tab 内容全宽，表格横向滚动  |
| 平板 | 同上                               | 同上                        |
| 桌面 | 卡片全宽（模板数量少，不需要网格） | Pipeline 横向流程图完整展示 |
