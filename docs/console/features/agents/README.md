# Agent 管理

查看和管理当前主题下注册的所有 Agent，展示实时状态，编辑 workspace 文件，查看任务历史与 Token 消耗。

## 路由

```
/agents           → Agent 列表页（卡片网格）
/agents/[id]      → Agent 详情页（概览 / 文件 / 任务 / 消耗）
```
