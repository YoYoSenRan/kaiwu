### 卡片结构

```
┌───────────────────────────┐
│  ● 状态灯  Agent 名称     │
│                            │
│  阶段    执行              │
│  模型    Claude Sonnet     │
│                            │
│  最后活跃  2 分钟前        │
└───────────────────────────┘
```

- 状态灯：online 🟢 + pulse 动画，idle 🟡，offline ⚫，error 🔴
- 点击卡片跳转详情页
- 禁用的 Agent 卡片半透明显示

### 响应式

| 断点 | 网格列数 | class                           |
| ---- | -------- | ------------------------------- |
| 手机 | 1 列     | `grid-cols-1`                   |
| 平板 | 2 列     | `md:grid-cols-2`                |
| 桌面 | 3-4 列   | `lg:grid-cols-3 xl:grid-cols-4` |

### 交互状态

| 状态         | 展示                                            |
| ------------ | ----------------------------------------------- |
| 加载中       | 卡片骨架屏（3-4 个占位卡片）                    |
| 空状态       | "尚未部署模板，请先在模板管理中部署" + 跳转链接 |
| 错误         | Error Boundary，显示错误信息 + 重试按钮         |
| Gateway 离线 | 卡片正常展示 DB 数据，状态灯统一显示 offline    |

### 响应式（详情页）

| 断点 | 差异                                 |
| ---- | ------------------------------------ |
| 手机 | 顶部信息栏折叠为两行，Tab 可横向滚动 |

### 详情页 Not Found

Agent ID 不存在时调用 `notFound()`，展示 "Agent 不存在" 提示。
