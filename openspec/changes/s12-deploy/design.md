## Context

试剑阶段已跑通（s11）。本阶段实现造物流的最后一步，让第一个物帖完整走完 6 阶段。

## Goals / Non-Goals

**Goals:**

- 器物成功部署到 Vercel
- 冒烟测试能检测到页面异常
- 复盘在 7 天后正确触发
- 属性计算正确
- 第一个物帖完整走完 6 阶段

**Non-Goals:**

- 不实现自动域名配置（手动配置）
- 不实现监控接入（属于打磨阶段）

## Decisions

### D1: 部署到 Vercel

通过 Git push 触发 Vercel 自动部署。鸣锣 Agent 负责构建打包 + push + 等待部署 + 冒烟测试。

### D2: 冒烟测试 — HTTP HEAD

简单的 HTTP HEAD 请求关键页面，检查返回 200。不做视觉回归测试。

## Risks / Trade-offs

- **Vercel 部署失败**：网络问题或构建错误。→ 冒烟失败自动回退到试剑。
- **复盘数据不足**：7 天内可能没有足够的用户数据。→ 复盘志标注"数据不足，仅供参考"。
