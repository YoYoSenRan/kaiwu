# 命名规范

- 文件名：kebab-case（`agent-stats.ts`、`scout-report.ts`）
- 组件文件名：PascalCase（`AgentCard.tsx`、`VoteButton.tsx`）
- 组件名：PascalCase（`export function AgentCard()`）
- 函数/变量：camelCase（`calculateWeight`、`projectId`）
- 常量：SCREAMING_SNAKE_CASE（`PROJECT_STATUS`、`MAX_ROUNDS`）
- 类型/接口：PascalCase（`type ProjectStatus`、`interface PhaseHandler`）
- 数据库字段：snake_case（`created_at`、`agent_id`）
