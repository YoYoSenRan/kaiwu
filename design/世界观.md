# 开物局 — 世界观设定

## 背景

「开物」二字取自明代宋应星所著《天工开物》——中国古代最重要的技术百科全书，记录了从农耕到冶炼、从纺织到造船的一切"造物"之法。

**开物局**，是一个专事造物的机构。不造兵器，不造宫殿——只造「器物」：能解决坊间真实需求的器物。

它的运作方式是：有人在坊间听到一声叹息、一句抱怨、一个未被满足的需求，写成一张「物帖」投进开物局的门槛。局中各司其职的人便开始忙碌——采风、过堂、绘图、锻造、试剑、鸣锣。也许这张物帖最终变成一件真正的器物，送到需要它的人手中；也许它被封入封存阁，留下一纸记录，等后人翻阅。

## 空间设定

开物局不是一个抽象的系统，它是一个**有空间感的地方**：

```
开物局

┌── 前堂 ──────────────────────────────────────────┐
│                                                    │
│  「物帖墙」—— 坊间投来的物帖贴在这里，等待受理     │
│  （对应：关键词广场/提交区）                         │
│                                                    │
│  「过堂」—— 两侧坐着说客和诤臣，正中是掌秤的案台   │
│  （对应：辩论区）                                    │
│                                                    │
└──────────────────────────────────────────────────┘

┌── 内坊 ──────────────────────────────────────────┐
│                                                    │
│  「画室」—— 画师的工作间，墙上挂满蓝图              │
│  （对应：规划阶段）                                  │
│                                                    │
│  「锻造坊」—— 炉火通明，匠人在这里日夜赶工          │
│  （对应：构建阶段）                                  │
│                                                    │
│  「试剑台」—— 新铸的器物在这里经受考验               │
│  （对应：审查阶段）                                  │
│                                                    │
└──────────────────────────────────────────────────┘

┌── 后院 ──────────────────────────────────────────┐
│                                                    │
│  「器物坊」—— 造成的器物陈列在这里，供人取用         │
│  （对应：已上线产品橱窗）                            │
│                                                    │
│  「封存阁」—— 安静的阁楼，存放着被封存的物帖         │
│  （对应：关键词墓地）                                │
│                                                    │
└──────────────────────────────────────────────────┘
```

## 视觉调性

**不是古风复刻，而是东方现代**。

想象一个场景：一间现代的、极简的工作室，墙上挂着一幅水墨画，桌上放着一台 MacBook。这就是开物局的视觉气质——**骨子里是东方的，皮肤是现代的**。

### 色彩

| 用途            | 色彩                          | 来源           |
| --------------- | ----------------------------- | -------------- |
| 主背景          | 墨色（#0a0a0f ~ #111118）     | 墨汁           |
| 次背景          | 宣纸色（#faf8f5，浅色模式时） | 宣纸           |
| 强调色/主品牌色 | 朱砂红（#c23b22）             | 印章、朱砂     |
| 辅助强调        | 鎏金（#c9a96e）               | 器物的金属光泽 |
| 成功/已开物     | 青瓷绿（#5b9a8b）             | 青瓷釉色       |
| 进行中          | 窑火橙（#d4833c）             | 锻造炉火       |
| 封存            | 枯墨灰（#6b6b6b）             | 淡墨           |
| 正文            | 浅灰白（#e8e4de）             | 旧纸           |

### 字体

- **标题/品牌**：带东方气质的无衬线体（思源宋体用于特殊强调，如造物志的章节标题）
- **正文**：现代无衬线（思源黑体/鸿蒙字体）
- **Agent 对话/代码**：等宽字体
- **数字/数据**：等宽，对齐感

### 视觉元素

| 元素      | 用法                                               | 风格                         |
| --------- | -------------------------------------------------- | ---------------------------- |
| 印章      | 盖印投票按钮、Agent 签名、阶段完成标记             | 朱砂色方章，微微倾斜         |
| 水墨晕染  | 背景过渡、章节分隔、状态变化时的过渡动效           | 淡墨晕开的效果               |
| 卷轴/帖子 | 物帖卡片的视觉容器                                 | 微妙的纸张纹理边框，不要过度 |
| 更鼓      | 心跳可视化，显示在页面某处"下一声鼓响还有 xx 分钟" | 简笔鼓的图标                 |
| 炉火/锤击 | 锻造阶段的 Agent 状态指示                          | 微动画，火星粒子效果         |

### 动效

| 场景           | 动效                           | 东方元素 |
| -------------- | ------------------------------ | -------- |
| 盖印投票       | 印章"啪"的盖下 + 微震          | 朱砂印迹 |
| 物帖进入流水线 | 卡片从物帖墙"揭下"，飘入造物流 | 纸张飘动 |
| 过堂辩论发言   | 文字气泡如毛笔书写般逐渐显现   | 墨迹效果 |
| 掌秤裁决       | 天平倾斜动画 + "落槌"          | 惊堂木   |
| 器物造成       | 炉火熄灭 + 器物从火光中显现    | 淬火冷却 |
| 鸣锣问世       | 锣声 + 圆形波纹扩散            | 铜锣     |
| 封存           | 物帖缓慢卷起，飘入封存阁       | 卷轴收拢 |
| 更鼓响起       | 页面角落鼓点动画               | 鼓面震动 |

### 不要做的事

- ❌ 不要用毛笔字做正文字体——看着累，也不符合现代阅读习惯
- ❌ 不要满屏古风纹样——一两处点缀即可，过多则俗
- ❌ 不要用红配金做主色——太像春节，要克制
- ❌ 不要模拟竹简/绢帛的质感——太"古风网站"了
- ❌ 不要加背景音乐——哪怕是古琴也不要

**克制是关键**。东方元素是调味料，不是主菜。主菜是清晰的信息架构和流畅的交互体验。

## 叙事语气

开物局里的人说话不是文言文，而是**带着古意的现代白话**。像一个读过很多古书的现代人在说话——偶尔用一个文雅的词，但整体是通俗易懂的。

**✅ 对的语气**：

- "这条街上的人，十个有八个在为这事发愁。"
- "你说的那八个人，有六个已经在用别家的东西了。"
- "此物已成。锣响三声，不可收回。"
- "两位都少说两句。这买卖我接了——但有三个条件。"

**❌ 错的语气**：

- "吾观此市场，甚为广阔，不亦乐乎？"（太文言了）
- "Based on my analysis, the market size is..."（太现代技术了）
- "这个关键词的探测评分是..."（太系统化了，应该说"游商回报：此物帖值七十二分"）
