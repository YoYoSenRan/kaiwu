# 开物局 — 设计文档

> 天工开物，每帖必应。
>
> 你写了一张物帖，投进开物局。游商采风、说客与诤臣过堂、掌秤裁决、
> 画师绘图、匠人锻造、试剑验器、鸣锣问世。
> 也许成器，也许封存——但每一张物帖，都会被认真对待。

## 一句话概念

一张物帖投进开物局。没人知道它的命运——可能在过堂上被诤臣一句话钉死，也可能过五关斩六将，锣响三声，器物问世。

造成的器物有造物志，封存的物帖有封存辞。开物局里的八个人不是工具，是有脾气、有成长、会争吵的角色。每造出一件新器物，网站多一篇造物志；每封存一张物帖，封存阁多一条值得后人翻阅的记录。
