# 造物志（/stories + /stories/:id）

## 页面定位

造物志是开物系统的核心叙事载体。列表页展示所有物帖的状态全景，详情页将每个创意从投帖到面世（或封存）的全过程编排为章回体叙事——让用户像读故事一样，看一个想法如何被 Agent 们协作打造成产品。

## 详情页 /stories/:id

### 整体结构

详情页采用章回体叙事结构，自上而下线性排列。每个章节对应物帖生命周期中的一个阶段，未到达的阶段不渲染。

### 章节标题统一格式

```
「 第 一 章 · 发 现 」
```

| 属性   | 值                                                 |
| ------ | -------------------------------------------------- |
| 字体   | `--font-display text-2xl md:text-3xl`              |
| 字间距 | `tracking-[0.15em]`                                |
| 对齐   | `text-center`                                      |
| 分隔符 | 居中圆点 `·`，前后空格                             |
| 装饰   | 上下各一条 `w-16 h-px bg-border mx-auto my-4` 细线 |
| 格子符 | `「」` 包裹，`text-muted-foreground`               |

| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| `< md`（移动端） | 列表卡片 `grid-cols-1`；详情页气泡 `max-w-[95%]`；封面概览条 `flex-wrap`，每行 2-3 项；章节标题 `text-xl`；commit 列表增删行数隐藏 |
