## 实时活动条（Live Activity Bar）

全局悬浮在底部，有活跃事件时自动出现：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ⚡ 诤臣刚在「AI 写作助手」过堂中说了一句狠话                    查看 →  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

位置: fixed bottom-4, left-1/2 transform -translate-x-1/2
宽度: max-w-xl
样式:
  bg: --card
  border: 1px solid --border
  shadow: --shadow-lg
  radius: --radius-lg
  padding: 12px 20px
  font: body-sm

动效:
  进入: slide-up from bottom + fade, 400ms
  离开: fade-out, 300ms
  自动消失: 8 秒后

行为:
  点击 → 跳转到对应页面
  右侧 ✕ 关闭
  SSE 推送新事件时更新内容
  无事件时不显示

频率控制:
  同时最多显示 1 条
  新事件到来时替换旧的（fade 过渡）
  冷却期: 同一项目的事件间隔 ≥ 30 秒
```

## 页面过渡

```
路由切换:
  Next.js App Router 自带的 loading.tsx 骨架屏
  不做全页 fade 过渡（避免卡顿感）
  同一页面内的 tab/filter 切换用 fade 200ms

滚动行为:
  路由切换后 scrollTo(0, 0)
  锚点跳转 scroll-behavior: smooth
  返回上一页恢复滚动位置
```
