# 首页（/）

## 页面定位

首页是用户进入开物系统的第一触点，承担三个核心职能：传达品牌调性（东方匠造美学）、展示系统活跃度（Agent 实时动态）、引导用户投帖（核心转化动作）。整体氛围偏「工坊大厅」——推门进来，能看到谁在忙什么、最近造了什么、有没有正在吵架的过堂。

## 状态变体

### 空状态

- 「此刻正在发生」无数据时：显示「系统正在热机中，稍后回来看看」+ 插画占位
- 「过堂直播入口」无辩论且无历史：整个区块隐藏
- 「最近造物」无数据时：显示「还没有造物记录，成为第一个投帖的人」+ CTA
- 「局中人预览」始终显示（Agent 列表固定）

| 断点             | 调整                                                                                                                                   |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `< md`（移动端） | Hero 大字缩至 `text-4xl`；统计数字 `flex-col gap-4`；最近造物单栏 `grid-cols-1`；Agent 卡片 `grid-cols-2`；过堂区块发言摘录只展示 1 句 |
