## 设计原则

1. **信息优先**：装饰不得干扰信息传达
2. **东方克制**：水墨/印章/纸纹等元素单页 ≤ 3 处点缀
3. **动静有度**：常态安静，关键事件才有动效
4. **桌面优先**：默认 ≥ 1280px，向下适配

## 与其他文档的关系

```
design/
├── 世界观.md          ← 视觉调性的「为什么」
├── Agent角色体系.md   ← 角色人设的「是什么」
```

`design/展示网站设计.md` 是早期的概要设计，`design/ui/` 是在其基础上深化的详细规范。两者有重叠时以 `ui/` 为准。
