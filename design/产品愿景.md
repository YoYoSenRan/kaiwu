# 产品愿景

## 一句话

**"天工开物，每帖必应。"**

你写了一张物帖，投进开物局的门槛。游商出门采风，说客和诤臣在堂上吵得不可开交，掌秤一锤定音，画师落笔绘图，匠人闷头锻造，试剑人验其利钝。也许这张帖子最终变成一件器物，送到需要它的人手中；也许它被封入封存阁，留下一纸记录。但无论如何——每一张投进来的物帖，都会被认真对待。

## 它不是什么

| ❌ 它不是                | ✅ 它是                                        |
| ------------------------ | ---------------------------------------------- |
| 一个技术 Demo            | 一座活着的造物工坊，每天都有新的物帖被讨论     |

| 一份自动化流程的说明书   | 开物局里八个有脾气的人在你眼前争吵、造物、鸣锣 |
| 又一个 SaaS Landing Page | 每造出一件新器物就多一篇造物志的连续剧         |

## 故事内核

### 为什么有人会一直来看这个网站？

**悬念**。

一张物帖投进开物局后，没有人知道它的命运——包括我们自己。它可能在采风阶段就暴露出致命缺陷；可能在过堂上被诤臣一句话钉死；可能过五关斩六将后在试剑台上翻车；也可能一路通关，锣响三声，器物问世。

这种不确定性就是造物本身。而我们把这种不确定性完整地、实时地、透明地呈现给每一个来客。

**角色**。

开物局里的人不是冷冰冰的程序。说客会在过堂第三轮被数据打脸后沉默一轮，然后换一个角度猛攻回来。诤臣偶尔会被一个创意打动，说出"好吧，这次我没什么好反对的"——这句话比任何赞美都珍贵。匠人会在三更天提交代码时附一句"这个问题折磨了我两个时辰"。

来客会开始"追"这些人，就像追一部剧的角色一样。

**封存**。

大多数展示网站只展示成功。开物局不一样——被封存的物帖、被掌秤否决的造物令，配上完整的过堂记录，是这个网站最有看头的内容。一张在过堂上被三轮驳回的物帖，其说客与诤臣的交锋精彩程度可能远超一件顺利造成的器物。封存阁不是废纸篓，是档案馆。

## 三种访客，三种体验

### 🍿 看客型——"我就来看看热闹"

**他们要什么**：有趣、好看、不用动脑

**我们给什么**：

- 首页就有实时发生的事："此刻，诤臣正在堂上驳一张社交健身的物帖"
- 过堂辩论的精彩片段，像短视频一样可以独立消费
- 封存阁——每张被封存的物帖都有掌秤写的封存辞
- 造物志的"快进版"——3 分钟读完一件器物从物帖到问世的全过程

**留存钩子**：订阅某个正在进行中的项目，"结果出来时通知我"

### 🛠 创造者型——"我也有个想法"

**他们要什么**：提交自己的关键词，看看 AI 怎么评价

**我们给什么**：

- 极低门槛的输入框："输入一个关键词，看看会发生什么"
- 即使不进入正式流水线，也能获得一份快速的市场扫描结果
- 如果他们的关键词被选中进入正式流水线，他们会成为这个造物志的"创始人"
- 最终产品上线时，致谢页面会有他们的名字

**留存钩子**：看到别人提交的关键词正在被辩论，产生"我也想试试"的冲动

### 🔬 研究者型——"这技术怎么实现的"

**他们要什么**：技术细节、架构设计、Agent 协作协议

**我们给什么**：

- 完整的技术架构文档，不藏着掖着
- 每个 Agent 的 Prompt 设计理念（不公开完整 Prompt，但公开设计思路）
- Agent 之间的消息流可视化——能看到底层是怎么协作的
- 开源的组件和模式，可以在自己的项目中复用

**留存钩子**：技术博客定期更新，复盘每个项目中遇到的技术挑战

## 内容飞轮

```
  用户提交关键词
       │
       ▼
  Agent 探测 + 辩论 ──→ 产生精彩内容（辩论、报告、决策过程）
       │
       ▼
  内容发布到展示网站 ──→ 吸引更多访客
       │
       ▼
  更多访客提交关键词 ──→ 更多 Agent 工作 ──→ 更多内容
       │
       ▼
  部分项目上线 ──→ 新产品本身又带来用户 ──→ 回流到展示网站
```

每一个关键词，无论成功还是失败，都是一条内容。网站不需要专门"运营内容"，流水线本身就在源源不断地生产内容。

---

## 情感设计原则

1. **制造牵挂**：让访客关心一个正在进行中的项目，想知道它的结局
2. **制造惊喜**：Agent 偶尔说出意想不到的话，辩论出现反转
3. **制造参与感**：提交关键词、订阅项目、看到自己的关键词被讨论
4. **制造仪式感**：新产品上线时有"发布仪式"，关键词被否决时有"告别仪式"
5. **制造共鸣**：失败不是耻辱，被否决的创意有体面的归宿和完整的记录

## 成功标志

### 第一层：它活着

- [ ] 第一个关键词完整走完 6 阶段，产出可访问的产品
- [ ] 展示网站上有 10+ 个造物志（含被封存的）
- [ ] Agent 辩论过程在网站上可回放

### 第二层：它有观众

- [ ] 有人自发在社交媒体分享 Agent 辩论的截图
- [ ] 有用户主动提交关键词
- [ ] 有人"订阅"了一个进行中的项目，等着看结果

### 第三层：它有灵魂

- [ ] 有人说"我最喜欢诤臣，它说得太对了"——局中人有了粉丝
- [ ] 有人翻遍封存阁，觉得"这些封存辞比成功故事更有价值"
- [ ] 有人反复来看，因为"想知道今天又发生了什么"——网站有了留存
