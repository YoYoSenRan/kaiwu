## 状态变体

| 状态               | 表现                                                                            |
| ------------------ | ------------------------------------------------------------------------------- |
| 加载中             | 骨架屏：卡片区域 `animate-pulse` 灰色占位块，雷达图用 `animate-spin` 的圆形占位 |
| 空状态（无 Agent） | 居中插图 + "尚无局中人入局"（`text-muted-foreground`）                          |
| 错误               | `error.tsx` 捕获，显示错误信息 + 重试按钮                                       |
| Agent 休眠态       | 卡片整体 `opacity-60`，状态 badge 显示"休眠"                                    |

## 响应式

| 断点             | 适配                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------- |
| `< md`（移动端） | 卡片 `grid-cols-2`，详情页头像缩小到 `w-40 h-52`，雷达图独占一行，布局全部 `flex-col` |
| `md`（平板）     | 卡片 `grid-cols-3`（可选）或保持 `grid-cols-4`，详情页左右布局                        |
| `lg`（桌面）     | 完整 `grid-cols-4` 布局，详情页最大宽度 `max-w-5xl mx-auto`                           |
| 关系图谱         | 移动端改为列表模式展示关系，不显示 SVG 图谱                                           |
