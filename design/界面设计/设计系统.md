# 开物局 · 设计系统

> 骨子里是东方的，皮肤是现代的。克制是关键——东方元素是调味料，不是主菜。

---

## 设计哲学

### 三条铁律

1. **信息优先**：任何装饰不得干扰信息传达。先确保用户看得懂，再考虑好不好看
2. **东方克制**：水墨、印章、宣纸纹理等元素单页不超过 3 处点缀。多一处则俗
3. **动静有度**：常态安静，关键事件才有动效。不要让页面"一直在动"

### 视觉气质参考

> 一间现代极简的工作室，墙上挂着一幅水墨画，桌上放着一台 MacBook。
> 窗外是竹影，桌上有一方印章。但房间里用的是 San Francisco 字体和 8px 网格。

                   → 用于：品牌标题、造物志章节名、角色名号、特殊强调

## 东方元素库

> 这是"调味料"，使用必须克制。

### 印章（Seal）

```
用途: 盖印投票、Agent 签名、阶段完成标记、掌秤裁决
风格: 朱砂色方形，边缘不规则（CSS clip-path 或 SVG mask）
大小: 24x24 (行内) / 40x40 (投票按钮) / 64x64 (裁决)
倾斜: rotate(-3deg ~ 3deg) 随机微偏
动效: 盖下时 scale 从 1.3→1 + opacity 从 0→1，时长 200ms

CSS 实现思路：
  .seal {
    color: var(--cinnabar);
    border: 2px solid currentColor;
    border-radius: 2px;
    padding: 2px 6px;
    font-family: var(--font-display);
    font-size: 12px;
    transform: rotate(-2deg);
    /* 边缘粗糙感用 SVG filter 或 mask-image */
  }
```

### 水墨晕染（Ink Wash）

```
用途: Hero 背景、章节过渡、页面间分隔
实现: radial-gradient 模拟，不用真实图片
频率: 每页最多 1-2 处

Hero 墨晕：
  background:
    radial-gradient(ellipse 600px 400px at 30% 20%, rgba(26,26,37,0.6), transparent),
    radial-gradient(ellipse 400px 600px at 70% 80%, rgba(26,26,37,0.4), transparent),
    var(--background);

章节分隔墨晕：
  height: 120px
  background: linear-gradient(to bottom, transparent, var(--muted) 50%, transparent)
  opacity: 0.3
```

### 纸纹质感（Paper Texture）

```
用途: 物帖卡片背景、造物志容器
实现: CSS noise filter 或 极淡的 SVG noise pattern
opacity: 3-5%，绝不能影响文字可读性

不要: 用真实纸张照片做背景——太重了
```

### 更鼓（Heartbeat Indicator）

```
用途: 全局，显示在导航栏或页面角落
样式: 小圆点 + 文字 "下一声更鼓 · 12:47"

  ● 12:47   ← 圆点有脉冲动画（box-shadow pulse），文字用 mono 字体

脉冲动画:
  @keyframes drum-pulse {
    0%, 100% { box-shadow: 0 0 0 0 var(--kiln); opacity: 1; }
    50% { box-shadow: 0 0 0 6px transparent; opacity: 0.6; }
  }
  animation: drum-pulse 2s ease-in-out infinite
```

## 动效规范

### 原则

1. **有意义**：每个动效对应一个用户认知事件，不是为了好看
2. **不打断**：动效时长不超过 400ms（过渡）或 600ms（强调）
3. **可降级**：`prefers-reduced-motion: reduce` 时关闭所有装饰性动效

### 过渡动效（Transition）

```
名称            属性               时长      曲线                 用途
─────────────────────────────────────────────────────────────────────────
t-fast          all                150ms     ease                 hover色变
t-base          all                200ms     ease                 按钮、badge
t-smooth        all                300ms     cubic-bezier(0.4,0,0.2,1)  卡片升起
t-slow          all                400ms     cubic-bezier(0.4,0,0.2,1)  展开收起
```

### 关键帧动效（Keyframe）

```
名称            描述                        时长    用途
─────────────────────────────────────────────────────────────────────────
fade-in         opacity 0→1                 300ms   元素出现
slide-up        translateY(8px)→0 + fade    300ms   卡片、列表项进入
scale-in        scale(0.95)→1 + fade        200ms   模态框、下拉菜单
stamp           scale(1.3)→1 + fade         200ms   盖印按钮
ripple          圆形波纹扩散                600ms   鸣锣效果
pulse-glow      box-shadow 脉冲             2000ms  进行中状态
ink-spread      径向渐变从 0%→100%          800ms   墨晕过渡（仅首页）

辩论气泡入场:
  说客: slide-in from left + fade, 300ms
  诤臣: slide-in from right + fade, 300ms
  掌秤: scale-in from center, 400ms, delay 200ms
```

### 滚动动效

```
实现方式: CSS scroll-driven animations 或 Intersection Observer
触发位置: 元素进入 viewport 底部 20% 时触发

进场效果（所有卡片/区块统一）：
  opacity: 0 → 1
  transform: translateY(16px) → 0
  transition: 400ms ease-out

注意: 不要用 parallax——东方美学讲究"静中有动"，parallax 太吵了
```

## 响应式断点

```
桌面优先，向下适配：

断点名    宽度          布局调整
─────────────────────────────────────────────────────
desktop   ≥ 1280px      默认，max-w-7xl 居中
laptop    ≥ 1024px      卡片网格从 4 列→3 列
tablet    ≥ 768px       卡片 2 列，导航折叠为汉堡菜单
mobile    < 768px       单列，字号适当缩小

关键适配点:
  - 角色卡片列阵: 4→3→2→1 列
  - 辩论气泡: 桌面左右对列 → 移动端上下堆叠
  - 关系图谱: 桌面交互式 → 移动端简化为列表
  - 雷达图: 桌面 320px → 移动端 240px
  - Hero 区: 桌面 hero 字号 → 移动端 h1 字号
```

## 暗色/浅色模式

```
默认: 暗色模式（开物局的墨色基调更有氛围感）
切换: 导航栏提供 toggle，用 next-themes 管理
策略: CSS 变量自动切换，组件不需要判断 dark:

注意:
  - 浅色模式下东方元素要更克制（宣纸底色本身就是"风格"）
  - 暗色模式下墨晕、印章效果更明显
  - 两种模式下对比度都要达到 WCAG AA (4.5:1)
```

## 无障碍

```
- 色彩对比度: 正文 ≥ 4.5:1，大标题 ≥ 3:1
- 焦点环: 2px solid var(--cinnabar)，offset 2px
- 键盘导航: 所有交互元素可 Tab 到达
- 屏幕阅读器: 装饰性 emoji 加 aria-hidden
- 动效: 尊重 prefers-reduced-motion
- 图片: 角色立绘提供 alt 文本
```
