```

  text-align: center

  font: --font-display, hero (48/56px), weight 700
  letter-spacing: 0.2em
  color: --foreground
  margin-bottom: 8px

  font: --font-display, body-lg (16px), weight 400
  color: --muted-fg
  letter-spacing: 0.1em
  margin-bottom: 48px

正文段落:
  font: body-lg (16px), line-height 2.0
  color: --foreground
  max-width: 560px
  margin: 0 auto
  text-align: left

  段落间距: 24px

  角色名（游商、说客等）:
    font-weight: 600
    color: 对应角色主色

  关键短语（"全自动运转"、"心跳"、"更鼓"）:
    color: --cinnabar
    font-weight: 500

最后一段 "八位局中人都是 AI Agent":
  padding-top: 24px
  border-top: 1px dashed --border
  font: body (14px), color --muted-fg
  用更小更低调的语气揭底
```

### 交互说明

- 无复杂交互，纯阅读
- 大标题进入视口时 fadeIn + 微弱 letterSpacing 收束动画（0.3em → 0.2em, 800ms）
- 角色名 hover 时显示对应角色 emoji 的 tooltip

### 响应式适配

```
桌面 (≥ 1024px): 如线框图
平板 (768-1023px): 同桌面，max-width 缩小到 480px
移动 (< 768px):
  大标题: 36px
  正文: body (14px), line-height 1.8
  max-width: 100%, padding 0 24px
```
